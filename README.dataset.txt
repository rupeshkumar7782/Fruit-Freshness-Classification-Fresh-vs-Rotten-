Fruit Freshness Dataset

This dataset contains curated images of apples, bananas, and strawberries categorized into two classes: fresh and rotten. The directory structure follows standard machine learning conventions to ensure compatibility with common training pipelines.

Directory Structure:
- Apple
   - fresh
   - rotten
- Banana
   - fresh
   - rotten
- Strawberry
   - fresh
   - rotten

Use Cases:
Image classification
Fruit freshness detection
Quality inspection and smart agriculture research

All images were collected from publicly available online sources and organized to support training, validation, and benchmarking of computer vision models.
